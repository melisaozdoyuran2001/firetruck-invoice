# firetruck-invoice  
Firetruck Invoice Generator

### Built With

- Next.js
- TypeScript
- Tailwind CSS
- @react-pdf/renderer

### Features

- Extracts truck info from listing URL
- Fetches data from the Garage backend API
- Dynamically generates a professional PDF invoice
- Allows users to download the invoice PDF
